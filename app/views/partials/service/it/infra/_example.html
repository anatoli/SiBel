Реализация:   Услуги ИТ-аутсорсинга для «АСД-Техника»

Клиент «АСД-Техника»


Компания по разработке технологий и производства оборудования для изготовления изделий из полимерных материалов.


Цель


Стабильная работа предприятия независимо от смены системного администратора; передача работ по поддержке, обслуживанию и модернизации ИТ-инфраструктуры авторитетной компании на долгосрочный период.


Задачи «СиБел»


Оказание услуг по технологическому сопровождению программного оборудования, вычислительной и офисной технике — аутсорсинг ИТ-функций.


Результат


Налаженная работа ИТ-инфраструктуры предприятия, мониторинг ее состояния, подтверждаемый ежемесячными отчетами специалистов.


Этапы работ по оказанию услуг ИТ-аутсорсинга для «АСД-Техника»

  1. Модернизация и техническое сопровождение средств вычислительной техники.

  2. Резервное копирование информации.

  3. Настройка и сопровождение оборудования и программного обеспечения на рабочих местах.

  4. Консультация пользователей по вопросам функционирования средств вычислительной техники.

  5. Мониторинг и устранение неисправностей ИТ-инфраструктуры предприятия.

  6. Подготовка ежемесячных актов сдачи / приемки работ / услуг.
